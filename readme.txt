Purpose of this project is to demonstrate an FSK-SIM technique in baseband for Optical Wireless Communication system.
This demonstration is used for the thesis "Hierarchical Subcarrier Intensity Modulation For Resilient Optical Wireless Communication", submitted to the University of Edinburgh on November 10, 2024 by Egecan Guler.

This project bases the Vivado layout and control of the PYNQ environment on "https://github.com/strath-sdr/rfsoc_radio".
